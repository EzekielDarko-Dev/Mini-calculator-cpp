# Mini Calculator (C++)

## Description
This project is a simple console-based calculator built using C++. It was developed as part of my programming practice while studying Electrical and Electronic Engineering.

The calculator uses a menu-driven interface and a loop structure to allow the user to continuously perform calculations until they choose to exit.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Menu-based user interface
- Continuous operation using loops
- Basic input handling

## Concepts Practiced
- Functions
- Conditional statements (if/switch)
- Loops
- User input/output
- Basic program structure
- Problem solving and debugging

## How it works
The program displays a menu of operations. The user selects an operation, enters two numbers, and the calculator performs the selected operation. The program continues running until the user chooses the exit option.
